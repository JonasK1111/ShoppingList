# Shopping List Project

## Starten

### Option A: direkt als Modul starten
```bash
python -m shopping_list.main
```

### Option B: Datei ausführen (wenn du im Paket-Ordner bist)
```bash
python shopping_list/main.py
```

## Struktur
- `shopping_list/` enthält den Source Code (Paket)
- `tests/` enthält (optionale) Unit-Tests

## Export/Import
Die App kann Listen als `.txt` exportieren und wieder importieren.
